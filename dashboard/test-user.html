Установка тестового пользователя

Откройте консоль браузера (F12) и выполните следующий код:

// Установка тестового пользователя в localStorage
localStorage.setItem('telegram_user', '{"id": 123456789, "first_name": "\u0422\u0435\u0441\u0442", "last_name": "\u041f\u043e\u043b\u044c\u0437\u043e\u0432\u0430\u0442\u0435\u043b\u044c", "username": "testuser"}');
console.log('Тестовый пользователь установлен:', JSON.parse(localStorage.getItem('telegram_user')));
